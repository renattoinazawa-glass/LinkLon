## LinkLon - Agência de Performance e Web Design

Este é o repositório da Landing Page oficial da **LinkLon**, focada em gestão estratégica de tráfego pago (Meta Ads e Google Ads) e desenvolvimento web de alta conversão.

## 🚀 Tecnologias Utilizadas
* **HTML5 / CSS3**
* **JavaScript** (Interatividade e Lógica)
* **Tailwind CSS** (Estilização e Responsividade)
* **SVG** (Ícones leves e escaláveis)

## 💡 Diferenciais do Projeto
* **UX/UI Design:** Focado em conversão e experiência do usuário.
* **Performance:** Código otimizado para carregamento rápido.
* **Acessibilidade:** Seguindo boas práticas de estrutura semântica.
